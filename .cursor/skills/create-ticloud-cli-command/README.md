# Guide for human

It is the guide for human use. Do not read it as AI context.

## How to use

The create-ticloud-cli-command skill be triggered automatically whenever you want to create a new command.

For example, you can trigger it by

- 我希望创建一个 private link connection 相关的命令。
- 我希望创建一个 private link connection 相关的命令，包含 ticloud serverless private-link-connection create, ticloud serverless private-link-connection,ticloud serverless private-link-connection list,ticloud serverless private-link-connection get,ticloud serverless private-link-connection delete 命令。
